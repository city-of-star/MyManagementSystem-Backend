# 高校在线实习管理平台 — 接口文档

> 文档版本：v0.3  
> 约定：除登录外，请求头携带 `Authorization: Bearer <access_token>`。  
> 网关地址与端口以实际部署配置为准。  
> 实习业务网关前缀：**`/internship/**`**（`StripPrefix=1` 后进入实习服务）。  
> 命名说明：**Dto** 表示入参（请求体 / 查询对象），**Vo** 表示出参（响应视图）；日期时间采用 ISO-8601 字符串，如 `2025-03-28T10:00:00`。

---

## 1. 通用约定

### 1.1 统一响应 `ResultVo<T>`

| 字段 | 类型 | 说明 |
|------|------|------|
| code | int | 0 成功，非 0 业务错误码 |
| message | string | 提示信息 |
| data | T | 载荷，失败时可为 null |

### 1.2 分页查询基类 `PageQueryDto`（各模块分页入参继承或内联相同字段）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | int | 否 | 页码，默认 1 |
| pageSize | int | 否 | 每页条数，默认 10，最大建议 100 |
| sortField | string | 否 | 排序字段（白名单映射到表列） |
| sortOrder | string | 否 | `ASC` / `DESC`，默认 `DESC` |

### 1.3 分页结果 `PageResultVo<T>`

| 字段 | 类型 | 说明 |
|------|------|------|
| list | T[] | 当前页数据 |
| total | long | 总条数 |
| pageNum | int | 当前页 |
| pageSize | int | 每页条数 |
| pages | int | 总页数（可由服务端计算） |

### 1.4 枚举与字典（与库表一致）

**企业审核 `auditStatus`**：`PENDING` | `APPROVED` | `REJECTED`  

**岗位状态 `positionStatus`**：`DRAFT` | `PUBLISHED` | `CLOSED`  

**申请状态 `applicationStatus`**：`PENDING` | `APPROVED` | `REJECTED` | `CANCELLED` | `IN_PROGRESS` | `COMPLETED`  

**周志状态 `weeklyLogStatus`**：`SUBMITTED` | `APPROVED` | `REJECTED`  

**材料类型 `materialType`**：`AGREEMENT` | `SAFETY` | `OTHER`  

**材料审核状态 `materialStatus`**：`SUBMITTED` | `APPROVED` | `REJECTED`

## 2. 合作企业 `Enterprise`

### 2.1 接口列表

| 序号 | 说明 | 方法 | 路径 | 权限倾向 |
|------|------|------|------|----------|
| E1 | 分页查询 | GET | `/internship/enterprise/page` | 管理员、教务 |
| E2 | 下拉列表（已通过且启用） | GET | `/internship/enterprise/options` | 已登录（管理端选企业） |
| E3 | 详情 | GET | `/internship/enterprise/{id}` | 已登录 |
| E4 | 新增 | POST | `/internship/enterprise` | 管理员、教务 |
| E5 | 修改 | PUT | `/internship/enterprise/{id}` | 管理员、教务 |
| E6 | 逻辑删除 | DELETE | `/internship/enterprise/{id}` | 管理员、教务 |
| E7 | 审核 | PUT | `/internship/enterprise/{id}/audit` | 管理员、教务 |

**E1 查询参数**（继承 `PageQueryDto` 字段，外加）：

| 字段 | 类型 | 说明 |
|------|------|------|
| keyword | string | 模糊匹配企业名称、联系人 |
| auditStatus | string | 筛选审核状态 |
| status | int | 0 停用 / 1 启用，可选 |

**E1 响应** `ResultVo<PageResultVo<EnterpriseListItemVo>>`

**E2 查询参数**：`keyword`（可选）  
**E2 响应** `ResultVo<List<EnterpriseOptionVo>>`

**E3 响应** `ResultVo<EnterpriseDetailVo>`

**E4 / E5 请求体** `EnterpriseSaveDto`  
**E7 请求体** `EnterpriseAuditDto`  
**E4～E7 无 data 或** `ResultVo<Void>` / `ResultVo<Boolean>`

### 2.2 DTO

**EnterpriseSaveDto**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| enterpriseName | string | 是 | 企业名称 |
| creditCode | string | 否 | 统一社会信用代码 |
| contactName | string | 否 | 联系人 |
| contactPhone | string | 否 | 联系电话 |
| address | string | 否 | 地址 |
| intro | string | 否 | 简介 |
| remark | string | 否 | 备注 |

**EnterpriseAuditDto**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| auditStatus | string | 是 | `APPROVED` / `REJECTED` |
| auditRemark | string | 否 | 审核说明 |

**EnterpriseListItemVo**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | long | 主键 |
| enterpriseName | string | 企业名称 |

| contactName | string | 联系人 |
| contactPhone | string | 电话 |
| auditStatus | string | 审核状态 |
| status | int | 启用状态 |
| createTime | string | 创建时间 |

**EnterpriseOptionVo**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | long | 主键 |
| enterpriseName | string | 企业名称 |

**EnterpriseDetailVo**（在 ListItem 基础上扩展）

| 字段 | 类型 | 说明 |
|------|------|------|
| address | string | 地址 |
| intro | string | 简介 |
